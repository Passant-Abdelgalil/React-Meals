# React Meals


### Built with
- React Js
  - React State
  - React Context
  - Portals, Fragments, Refs
- CSS
  - CSS Modules
  - Drawing Icons
- Firebase

## Getting Started

### Installation

1. Clone the repository

   ```
   git clone https://github.com/Passant-Abdelgalil/coach-finder.git
   ```

2. Navigate to the directory

   ```
   cd coach-finder
   ```

3. Install dependencies

   ```
   npm i
   ```

### Compiles and hot-reloads for development

```
npm start
```
